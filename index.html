HTML

HTML, o HyperText Markup Language, es el lenguaje de marcado estándar para crear páginas web. Se utiliza para estructurar el contenido, definiendo elementos como encabezados, párrafos, enlaces e imágenes. Junto con CSS y JavaScript, es uno de los pilares fundamentales del desarrollo web.

CSS

CSS, o Cascading Style Sheets, es un lenguaje de estilo utilizado para describir la presentación de documentos HTML. Permite definir el diseño, colores, fuentes y el layout de una página web, mejorando su apariencia y experiencia visual.

Diseño Grafico

El diseño gráfico es la práctica de crear contenido visual para comunicar mensajes. Utiliza elementos como tipografía, color, imágenes y composiciones para transmitir ideas de manera efectiva. Se aplica en diversas áreas, como publicidad, branding, web y publicaciones, buscando captar la atención y transmitir emociones.
